# Dissertation-Thesis
  Targeting SARS CoV-2 Receptor Binding Domain-ACE2
  First we docked already known compounds on to the S-RBD and ACE2 region to check for the best docking model.
  We used Sars CoV 2 PDB model to dock  Chembridge library compounds which binds to S-RBD and ACE2 markers of the viral pritein capsule.
  Further we took the best performing compounds and did MD simulation of the compound to check for bond satblity inside human body.
